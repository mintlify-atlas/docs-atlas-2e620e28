---
title: "Appium UiAutomator2 Driver"
description: "Automate native, hybrid, and mobile web apps on Android devices using Google's UiAutomator2 framework."
---

Appium UiAutomator2 Driver is a test automation framework for Android devices. It automates native, hybrid, and mobile web apps on real devices and emulators, operating within the [W3C WebDriver protocol](https://www.w3.org/TR/webdriver/) with Android-specific extensions.

The driver is part of the [Appium](https://github.com/appium/appium) mobile automation ecosystem and proxies commands to the [UiAutomator2 server](https://github.com/appium/appium-uiautomator2-server), which uses Google's UiAutomator framework under the hood.

<Note>
  Since version 5.0.0, this driver is only compatible with Appium 3. Install it with `appium driver install uiautomator2`.
</Note>
