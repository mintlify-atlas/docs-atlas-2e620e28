title: "Quickstart"
description: "Install the UiAutomator2 driver and run your first Android automation test"
---

This guide walks you through installing the driver, preparing your environment, and running a simple test against an Android device or emulator.

<Steps>
  <Step title="Install Appium 3">
    Install Appium globally via npm. Appium 3 is required for this driver.

    ```bash
    npm install -g appium
    ```

    Verify the installation:

    ```bash
    appium --version
    ```
  </Step>

  <Step title="Install the UiAutomator2 driver">
    Use the Appium driver CLI to install the UiAutomator2 driver into your Appium distribution.

    ```bash
    appium driver install uiautomator2
    ```
  </Step>

  <Step title="Verify the installation">
    Confirm that the driver is listed as installed:

    ```bash
    appium driver list --installed
    ```

    You should see `uiautomator2` in the output with its version number.
  </Step>

  <Step title="Set up Android SDK and environment variables">
    The driver requires the Android SDK platform tools (`adb`) and a Java JDK to be installed and discoverable.

    1. Install [Android Studio](https://developer.android.com/studio) or the [Android SDK command-line tools](https://developer.android.com/studio/releases/platform-tools) to get `adb`.
    2. Install a JDK (Java 8 for Android API level < 30; Java 9+ for API level 30 and above).
    3. Set the required environment variables:

    | Variable | Purpose |
    | --- | --- |
    | `ANDROID_HOME` or `ANDROID_SDK_ROOT` | Path to the Android SDK installation directory |
    | `JAVA_HOME` | Path to the JDK installation directory |

    Example (Linux/macOS):

    ```bash
    export ANDROID_HOME=$HOME/Library/Android/sdk
    export JAVA_HOME=$(/usr/libexec/java_home)
    export PATH=$ANDROID_HOME/platform-tools:$PATH
    ```
  </Step>

  <Step title="Run doctor checks">
    The driver ships with a `doctor` command that validates your environment automatically:

    ```bash
    appium driver doctor uiautomator2
    ```

    Fix any issues reported before proceeding. The doctor checks both required prerequisites and optional extensions.
  </Step>

  <Step title="Connect a device or start an emulator">
    **Real device:** Enable [USB debugging](https://developer.android.com/studio/debug/dev-options) on the device, connect it via USB, and confirm it appears online:

    ```bash
    adb devices -l
    ```

    **Emulator:** Create and launch an AVD via Android Studio or `avdmanager`, then confirm it appears in the `adb devices -l` output.
  </Step>

  <Step title="Start Appium server">
    ```bash
    appium
    ```

    The server listens on `http://127.0.0.1:4723` by default.
  </Step>

  <Step title="Create a session">
    Write a test that opens a session with the `UiAutomator2` automation name. Examples in Python, Java, and JavaScript (WebdriverIO) are shown below.

    <Note>
      Replace `/path/to/your/app.apk` with the actual path to your application package, and adjust `deviceName` or `udid` to match your connected device.
    </Note>

    <CodeGroup>

    ```python python
    from appium import webdriver
    from appium.options.android import UiAutomator2Options

    options = UiAutomator2Options()
    options.platform_name = 'Android'
    options.device_name = 'emulator-5554'
    options.app = '/path/to/your/app.apk'
    options.no_reset = True

    driver = webdriver.Remote('http://127.0.0.1:4723', options=options)

    # Find an element and interact with it
    el = driver.find_element('accessibility id', 'Submit')
    el.click()

    driver.quit()
    ```

    ```java java
    import io.appium.java_client.AppiumBy;
    import io.appium.java_client.android.AndroidDriver;
    import io.appium.java_client.android.options.UiAutomator2Options;
    import java.net.URL;

    UiAutomator2Options options = new UiAutomator2Options();
    options.setDeviceName("emulator-5554");
    options.setApp("/path/to/your/app.apk");
    options.setNoReset(true);

    AndroidDriver driver = new AndroidDriver(new URL("http://127.0.0.1:4723"), options);

    WebElement el = driver.findElement(AppiumBy.accessibilityId("Submit"));
    el.click();

    driver.quit();
    ```

    ```javascript javascript
    const { remote } = require('webdriverio');

    const driver = await remote({
      hostname: '127.0.0.1',
      port: 4723,
      capabilities: {
        platformName: 'Android',
        'appium:automationName': 'UiAutomator2',
        'appium:deviceName': 'emulator-5554',
        'appium:app': '/path/to/your/app.apk',
      }
    });

    const el = await driver.$('~Submit');
    await el.click();

    await driver.deleteSession();
    ```

    </CodeGroup>
  </Step>
</Steps>
